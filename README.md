# web
The GitHub Pages website for LD-archive.org made using Jekyl

## About
LD-Archive.org is the home of archived content from The Lighting Design Archive,
a project of the Royal Academy of Dramatic Art (RADA) in London. The Archive contains a 
collection of lighting design materials submitted by lighting designers for live performance.

## Contributing to the code
We welcome contributions to the code. If however, you are looking to contribute to the archive
providing some of your materials, then please [visit our contributions page](https://ld-archive.org/contribute).

Contributions to the code are always welcomed.
