# web
The GitHub Pages website for LD-archive.org made using Jekyl. 
This project is a simple and accessible way of accessing, searching and 
viewing materials that are stored at our long term partner. 
This site doesn't store or monitor the records for accuracy. 

For more infomration about the community you can simply visit [the community here]().

## About
LD-Archive.org is the home of content from The Lighting Design Archive,
a project of the Royal Academy of Dramatic Art (RADA) in London. The Archive contains a 
collection of lighting design materials submitted by lighting designers for live performance.

## Contributing to the code
We welcome contributions to the code. If however, you are looking to contribute to the archive
providing some of your materials, then please [visit our contributions page](https://ld-archive.org/contribute).

Contributions to the code are always welcomed. You can do so by simply making use of GitHubs tools. You can [report an issue](https://github.com/ld-archive/ld-archive-web/issues) or if you know the fix, simply create new fork!
